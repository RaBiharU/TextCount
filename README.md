# TextCount

## 🤔これは何
Windowsでテキスト選択部分の文字数を表示するだけのツールです

MS Wordやブラウザツールを使わずに、任意のアプリケーションで文字数カウントできるため便利だと思います

## 🔧導入方法
### AutoHotKeyのインストール
AutoHotKeyを[ここ（AutoHotKey公式サイト）](https://www.autohotkey.com/)からダウンロード・インストール

### ahkファイルの実行
`TextCount.ahk`をダウンロード・クリックして実行

使用する際にはPC起動の度に実行する必要があります

### 文字数カウント
テキストを選択し、`Ctrl + Win + C`を押すと、文字数と改行を除いた文字数が表示されます

表示内容やショートカットキーは`TextCount.ahk`から自由に変更可能です
